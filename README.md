# Employee Salary Prediction

## Overview

Machine Learning project that predicts employee salaries based on:

- Age
- Gender
- Education Level
- Job Title
- Years of Experience

## Models Compared

- Linear Regression (R² = 0.852)
- Decision Tree Regression (R² = 0.792)
- Random Forest Regression (R² = 0.866)

## Best Model

Random Forest Regressor

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Run

pip install -r requirements.txt

streamlit run app.py
